Command Description

This description presents the commands used to control the flow of program execution, as well as their functional purpose.

set mark jump 1

Alternative notation: V 1

Description: Sets a mark at the current position in the program and jumps to the line designated as 1. This allows saving the current execution state and returning to it later.

set zero jump 1

Alternative notation: X 1

Description: Clears the mark if it exists and jumps to line 1. This command is used to reset the state if the mark is no longer needed.

move right jump 1

Alternative notation: -> 1

Description: Moves the pointer to the right and jumps to line 1. This allows navigating through the data structure or program, changing the current position.

move left jump 1

Alternative notation: <- 1

Description: Moves the pointer to the left and jumps to line 1. This is similar to the previous command but in the opposite direction.

if mark ? jump 2 : jump 1

Alternative notation: ? 1 ; 2

Description: Checks for the presence of a mark. If the mark is absent, the program jumps to line jump 1; otherwise, it jumps to line jump 2. This command allows implementing conditional logic in the program.

if zero ? jump 2 : jump 1

Alternative notation: ? 2 ; 1

Description: Similar to the previous command, it checks for the presence of the "zero" mark. Depending on the result of the check, the program jumps to one of the two lines.

stop

Description: Ends the execution of the program. This command is used to stop all processes and exit the program.

General Description of Commands

These commands allow controlling the flow of program execution, setting and clearing marks, as well as implementing conditional jumps. They are essential tools for creating logic in programming.

Additional Information

Each empty line or comment is considered a blank command and has its number in the command sequence corresponding to the line on which it is located. This helps maintain order and structure in the program, even if there are inactive or explanatory elements.
